# Developer Command

## Usage:
`ad!dev shutdown` - Shutdown.



`ad!dev premium user <@mention/id> [remove]` - Set user as Premium.

`ad!dev premium guild <@mention/id> [remove]` - Set guild as Premium.



`ad!dev <eval/run> <engine> <code>` - Evals a piece of code.



`ad!dev <enablecallsite/disablecallsite>` - Callsite tracing.



`ad!dev weebsh` - Weeb.sh Dump.

`ad!dev weebsh -account` - Weeb.sh Account Dump.

`ad!dev weebsh <[-type <...>] [-tags <...>] [-ext <...>] [-hidden <...>] [-nsfw <...>]>` - Gets a random Weeb.sh image.



`ad!dev peek nowplaying` - Peek musics playing.



`ad!dev reloadassets` - Reload assets.

`ad!dev genemotelist` - EmotesKt dump.

`ad!dev gencmdyml` - Generate commands.yml file.

`ad!dev <gencmdhtml/gencmdmd>` - Generate snipppet. (botlists)
